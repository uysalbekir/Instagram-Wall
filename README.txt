How to use:

(Warning: With "Sandbox Mode" you are only able to load images from your own Instagram account due to restrictions!)
AJAX calls can be filtered by altering the URL. Go to https://www.instagram.com/developer for more information.

var token = enter the access token provided by Instagram
num_photos = quantity of pictures to be loaded
userid = this can be changed to hashtag or other endpoints. For more info: https://www.instagram.com/developer/endpoints
